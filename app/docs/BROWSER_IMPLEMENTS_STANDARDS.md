# 浏览器实现了所有标准，前端只是使用者

## 核心概念

```
W3C 标准 = 规则书
浏览器 = 规则的实现者（最完美、最兼容）
前端 = 规则的使用者

前端写的代码 = 使用浏览器已经实现的功能
浏览器已经处理了所有的复杂性
前端只需要调用浏览器提供的接口
```
